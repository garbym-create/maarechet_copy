# מתכנן מערכת השעות 🗓️

כלי חופשי לתכנון ובנייה של מערכת השעות הבית-ספרית.

## יכולות
- לוח כיתות ולוח מורים מסונכרנים — מקלידים שיבוץ פעם אחת והוא מופיע בכל התצוגות
- שיעורים משותפים, מקבצים וחלוקה לקבוצות (מתקדמים/בסיס, מגמות, מורה לכל כיתה)
- זיהוי התנגשויות וימים חופשיים למורים
- מעקב מכסות מורים (פרונטלי+פרטני+שהות) ותקן שעות כיתתי לפי מקצוע
- שיוך תלמידים לקבוצות ודוח פיצולים
- שכפול שיבוצים ובלוקים
- הדפסות: סדין מלא ומערכות אישיות (עמוד לכל כיתה/מורה)
- גיבוי ושחזור בקובץ

## חשוב לדעת
הנתונים נשמרים **בדפדפן בלבד** — הורידו קובץ גיבוי (⬇) בסוף כל עבודה.

## הסרת אחריות
המערכת פותחה באופן אישי וניסיוני ומסופקת כפי-שהיא (AS-IS), ללא אחריות מכל סוג.
הנתונים נשמרים בדפדפן המשתמש בלבד ובאחריותו לגבותם.
